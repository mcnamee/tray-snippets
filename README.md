# Tray Snippets

A little text Snippets manager, built on ReactJS for:

- The web
- Your desktop menu bar/tray (using [Electron](http://electron.atom.io))

## Install

```sh
npm i
```

## Running

### In your browser

```sh
npm start
```

### OS Menu bar

```sh
npm run electron-start
```

## Packaging

### In your browser

```sh
npm run build
```

### OS Menu bar

```sh
npm run electron-package
```
